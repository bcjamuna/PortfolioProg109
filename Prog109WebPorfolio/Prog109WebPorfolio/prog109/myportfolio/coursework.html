Fall Quarter 2015

Prog109 ~ 5 credit class

Instructor: Maricel Medina-Mora

Description:
Class uses HTML, CSS and JavaScript. Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects.

After completing this class, students should be able to:

  • Develop Web Pages using html elements to code a template for a web page.
  • Use variables with appropriate data-types and scope.
  • Use appropriate arithmetic, logical and comparison operators to develop complex expressions.
  • Apply inline, embedded, and external cascading style sheets (CSS) utilizing relative and absolute positioning.
  • Optimize images for web page display
  • Instantiate and use objects (including web forms and controls) and create JavaScript functions.
  • Code event-driven procedures (event-handlers).
(from bellevuecollege.edu)

Reflection:

HTML

Building code to create functional websites.

The first couple weeks of class covered how to build HTML, forms, etc.

CSS

Adding CSS(Cascading Style Sheets) to style webpages.

Then two to three weeks were used to enhance HTML using CSS.

Javascript

Automated code designed to work in webpage background.

In the remaining month and a half Javascript was implemented. Some coursework of Javascript includes form validation, automatic retrieval of current date and time, and building shapes on a webpage using loops and user prompts.
